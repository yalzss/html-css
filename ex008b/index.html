O que é HTML?

HTML é a linguagem de marcação padrão usada para criar e estruturar páginas da web. Ela define a estrutura básica de um site, indicando o conteúdo e como ele deve ser exibido no navegador. HTML não é uma linguagem de programação, mas uma linguagem de marcação, pois utiliza "tags" (ou etiquetas) para identificar e estruturar diferentes elementos de uma página.

O que é Css?

CSS é uma linguagem de estilo utilizada para definir a apresentação visual de uma página web escrita em HTML ou XML. Ou seja, enquanto o HTML é responsável pela estrutura da página, o CSS cuida da aparência, como cores, fontes, layouts, tamanhos e outros aspectos visuais.
